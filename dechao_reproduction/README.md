# Dechao reproduction experiments

This directory contains MNIST two-sample testing reproduction experiments and CNN embedding MMMD experiments.

Tracked content includes:
- Experiment code and configuration files.
- Small result CSVs, summary CSVs, logs, and figures.
- CNN training metadata such as `train_config.json`, `train_history.csv`, `train.log`, and `training_curves.png`.

Intentionally not tracked:
- MNIST raw data files under `data/`.
- CNN model weights such as `mnist_cnn_checkpoint.pt`.
- R checkpoint files such as `checkpoint.rds`.
- Large per-resample diagnostics such as `*_sigma_diagnostics.csv` and `*_all_inner_iters.csv`.
- The interrupted partial 7-method two-null Type-I run.

The shared CNN checkpoint can be regenerated by running the experiment scripts; the saved GitHub results are sufficient for inspecting the reported power/type-I summaries and figures.
